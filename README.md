# SEU Labs Website V2

A polished multi-page static website for the SEU Labs faculty pilot.

## Files

- `index.html` — Home page
- `apply.html` — Demo intake form built into the website
- `process.html` — Dedicated process page
- `guidelines.html` — Project and security guidelines
- `resources.html` — Tools and templates page
- `projects.html` — Project directory page
- `support.html` — Support page
- `styles.css` — Apple-style responsive design
- `script.js` — Mobile menu and demo form behavior
- `assets/stedwards-logo.png` — St. Edward's University logo

## Upload update to GitHub

Copy these files into your local `seu-labs-upload` folder, commit, and push.

```bash
cd ~/seu-labs-upload
cp -R "$HOME/Desktop/JOB OIT/SEU-LABS WEBSITE/seu-labs-site-v2/"* .
git add .
git commit -m "Redesign SEU Labs site with dedicated pages and demo form"
git push origin main
```

GitHub Pages will rebuild automatically after the push.
